MULTIPLE SPOUTCAM INSTANCES
===========================

This folder contains 4 SpoutCam instances that can run simultaneously.

FILES:
------
SpoutCam1_64.ax  - SpoutCam1 (64-bit)
SpoutCam2_64.ax  - SpoutCam2 (64-bit) 
SpoutCam3_64.ax  - SpoutCam3 (64-bit)
SpoutCam4_64.ax  - SpoutCam4 (64-bit)

REGISTRATION:
-------------
Individual registration:
- SpoutCam1_register.bat - Register SpoutCam1 only
- SpoutCam2_register.bat - Register SpoutCam2 only  
- SpoutCam3_register.bat - Register SpoutCam3 only
- SpoutCam4_register.bat - Register SpoutCam4 only

Bulk registration:
- register_all.bat - Register all 4 instances at once

UNREGISTRATION:
---------------
Individual unregistration:
- SpoutCam1_unregister.bat - Unregister SpoutCam1 only
- SpoutCam2_unregister.bat - Unregister SpoutCam2 only
- SpoutCam3_unregister.bat - Unregister SpoutCam3 only  
- SpoutCam4_unregister.bat - Unregister SpoutCam4 only

Bulk unregistration:
- unregister_all.bat - Unregister all 4 instances at once

PROPERTIES/SETTINGS:
--------------------
Open settings dialog for each camera:
- SpoutCam1Properties.cmd - Configure SpoutCam1
- SpoutCam2Properties.cmd - Configure SpoutCam2
- SpoutCam3Properties.cmd - Configure SpoutCam3
- SpoutCam4Properties.cmd - Configure SpoutCam4

USAGE:
------
1. Right-click any registration .bat file and "Run as administrator"
2. Use SpoutCamSettings.exe to configure sender names and settings
3. Select SpoutCam1, SpoutCam2, SpoutCam3, SpoutCam4 in your applications

Each camera can connect to different Spout senders simultaneously!
